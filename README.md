# Gourmet au Catering

A modern responsive catering service website inspired by W3Schools Gourmet Catering Template.  
Built with HTML, CSS, Bootstrap & JavaScript.

## Features
- Fully responsive
- Smooth scroll animations
- Bootstrap layout
- Contact form
- Scroll-to-top button

## Technologies Used
HTML   | CSS  | Bootstrap 5 | JavaScript
